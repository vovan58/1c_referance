# Требования к программисту 1С
## Требования к опытному программисту
### Опыт
Не мене полутора лет программирования 1С

### Позиция
Умение самостоятельно определить путь решения задачи

Продумать структуру алгоритма

Создать метаданные для решения задачи

Выбрать способ обмена для решения задачи

Также важно умение взаимодействовать с людьми.

Уточнять задачу до тех пор, пока не станет понятно, какой результат заказчик хочет получить.

Задавать вопросы коллегам, чтобы получить консультацию
по неизвестной технологии. 

Как можно раньше рассказать команде, что требуется помощь решения
для того, чтобы уложиться в сроки? 

Хорошим подтверждением этого уровня квалификации являются
сертификаты уровня 1С специалист. Получите один или несколько. 

### Навыки владения профессиональными инструментами
#### Платформа

Встроенный язык программирования

Запросы

Интерфеисы и формы

СКД и отчеты

Интеграция : протоколы обмена, SOAP,  http, умение публикации базы на web-сервере

Инструменты для расследования проьлем : механизм отладки

Инструменты для решения узких задач: регистры бухгалтерии, расчета, бизнес процессы, внешние источники данных

#### Библиотека стандартных подсистем 
Какие возможности дает библиотека? Какие из них чаще всего применяются?
Обязательно знать механизмы длительных операций, работы
с отчетами и формирования печатных форм.
Применять программный интерфейс для вывода сообщений
пользователю, получения значения реквизита по ссылке,
а также получения ссылки на текущего пользователя
систем.

Желательно понимать принцип работы таких подсистем
как свойства, управление доступом, а также механизмов
обработчиков обновления.

В целом ориентироваться в документации подсистем
на ИТС, хотя бы один раз прочитать перечень подсистем
и список задач, решаемый каждый из них.

#### Стандарты разработки.
Обязательно хотя бы раз прочитайте целиком стандарты
разработки от фирмы 1С на сайте ИТС.
